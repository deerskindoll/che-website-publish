Get Started with Eclipse Che

Use Eclipse Che workspaces at codenvy.com or run Che on Windows, Mac or Linux.

How to Get Help

Issues: Sometimes the unexpected happens. If it does, post issues to our GitHub page. Please follow the issue reporting guidelines.

Documentation: We put a lot of effort into our docs. If there are improvements or errors, we'd love that feedback.

Chat: Che engineers hang out on Gitter, IRC and Slack.

  • Try Eclipse Che in the Cloud

    Codenvy hosts Eclipse Che workspaces with free accounts.

  • Download Eclipse Che

    Install binaries from an Eclipse mirror or build from source.
