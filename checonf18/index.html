The cloud workspace virtual event.

CheConf 2018 is coming February 21, 2018!

Register for CheConf

Our second Eclipse Che user and developer conference.

This one day virtual conference brings insights into how cloud developer workspaces are changing how applications are created, and how companies are building cloud-native developer tools.

See the future of cloud-native development.

Attend this one day virtual conference to hear contributors and maintainers from across the Eclipse Che and cloud development ecosystem discuss how containerized developer workspaces can drive better development workflows, team productivity and release frequency.

Topics areas for 2018 will include:

  • Eclipse Che use cases for teams.
  • Extending and customizing Eclipse Che.
  • Language Server Protocol (LSP) and language servers.
  • Using Eclipse Che in the Kubernetes ecosystem.
  • Cool technology for cloud development.

  • Insight
    Eclipse Che is bringing continuous practices to software development. Explore how Che works, and best practices to get the most from it in your team.

  • Education
    CheConf is your chance to go deep with live tutorials and sessions on how to use and extend Che and make the most of the language server protocol.

  • Extension
    Organizations around the world are extending Che to drive their own developer ecosystems. Learn their approaches and hear about their successes.
