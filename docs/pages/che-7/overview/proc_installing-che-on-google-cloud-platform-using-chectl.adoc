= Installing {prod-short} on Google Cloud Platform using {prod-cli}

.Prerequisites

* The `{prod-cli}` management tool is installed. See link:{site-baseurl}che-7/using-the-chectl-management-tool/[Installing the `{prod-cli}` management tool].

.Procedure

To install {prod-short} on Google Cloud Platform, run the following `{prod-cli}` command:
