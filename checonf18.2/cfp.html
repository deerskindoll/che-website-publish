Call for Speakers

CheConf brings together the Eclipse Che community, its user and surrounding ecosystem for a day long virtual conference covering topics that affect cloud development and containerization.

We are pleased to announce the next CheConf virtual event, this fall in 2018, to be held on December 5th 2018.

Submit a Session Proposal

Submitting a Session

The key to getting a session accepted is the abstract - be clear on what you'll cover and why it will benefit attendees. The abstract is your chance to really "sell" the value of attending your session.

Important dates

  • Nov 7: Deadline for submissions
  • Nov 12: Speakers notified and schedule published
  • Dec 5: CheConf 2018.2

    • We will be notifying everyone who submits a session regarding acceptance in January, 2018.

    CheConf 2018.2 is intended to focus on the Eclipse Che community, and what is new in Eclipse Che 7, including, but not limited to:

    • IDE products on top of Eclipse Che
    • Bringing Enterprise features to cloud IDEs
    • Language Server Protocol and Debug Adaptor Protocol implementations, and their integration into the IDE
    • Integration of developer tools with developer workspace management
    • Using a cloud IDE for use-cases like IOT and embedded software development
    • Extensibility of Eclipse Che and Theia — IDE plug-ins, Che server plug-ins
    • Improving the Eclipse Che or Theia community user and contributor experience

    We are now inviting proposals for talks about all of these topics.
